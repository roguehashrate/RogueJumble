# RogueJumble

An opinionated Nostr client with a mobile first UI and some customizaton.

Experience RogueJumble at [https://jumble.roguehashrate.com](https://jumble.roguehashrate.com)

## Run Locally

```bash
# Clone this repository
git clone https://github.com/roguehashrate/RogueJumble.git

# Go into the repository
cd RogueJumble

# Install dependencies
npm install

# Run the app
npm run dev
```
